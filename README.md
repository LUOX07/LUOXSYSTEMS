# LUOXSYSTEMS - Sitio Corporativo

Sitio estatico de LUOXSYSTEMS orientado a presentacion comercial, captacion de consultas y posicionamiento digital.

## Dominio actual de publicacion

- Produccion actual: `https://luox07.github.io/LUOXSYSTEMS/`

## Cerrar lanzamiento con dominio propio

1. Conecta tu dominio en GitHub Pages, Netlify o Vercel.
2. Reemplaza `https://luox07.github.io/LUOXSYSTEMS/` en `index.html`, `robots.txt` y `sitemap.xml` por tu dominio final.
3. Si usas GitHub Pages con dominio propio, agrega un archivo `CNAME` con el dominio exacto.
4. Vuelve a publicar para actualizar canonical, Open Graph y sitemap.

## Publicar en Netlify

1. Sube este proyecto a un repositorio en GitHub.
2. Entra en Netlify y elige **Add new site > Import from Git**.
3. Selecciona el repositorio.
4. Build command: (vacio)
5. Publish directory: `.`
6. Publica.

## Publicar en Vercel

1. Sube este proyecto a un repositorio en GitHub.
2. Entra en Vercel y elige **Add New > Project**.
3. Importa el repositorio.
4. Framework preset: **Other**.
5. Build command: (vacio)
6. Output directory: (vacio)
7. Deploy.

## Personalizacion rapida

- Nombre de marca: editar en `index.html`.
- Numero de WhatsApp y telefono: editar `595982912585` en `index.html`.
- Dominio canonico y preview social: editar la URL base en `index.html`.
- Colores y estilo: editar variables en `styles.css`.

## Estructura

- `index.html`: contenido principal
- `styles.css`: diseno
- `script.js`: menu movil, animaciones y formulario
- `site.webmanifest`: configuracion PWA e iconos
- `robots.txt`: rastreo para buscadores
- `sitemap.xml`: mapa del sitio
- `netlify.toml`: config para Netlify
- `vercel.json`: config para Vercel
